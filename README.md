# 合唱ステージ配置メーカー

合唱団・合唱イベント向けのステージ配置図作成ツールです。

各パートの人数、または氏名・パートの名簿から、複数の配置案を作成し、カードをドラッグして直感的に並べ替えることができます。  
作成した配置は、PNG画像、印刷/PDF、配置ファイルとして保存できます。

## 特徴

- SATB、S1/S2/A1/A2/T1/T2/B1/B2 などの多声部に対応
- カスタムパート設定に対応
- 2列〜6列の配置案を作成可能
- 配置パターンテンプレートに対応
  - パート別・縦縞
  - 女声前列 / 男声後列
  - 男声中央 / 女声両端
  - 混合配置
  - カスタムテンプレート
- 千鳥配置に対応
- PC・スマートフォンの両方で操作可能
- PNG画像として保存可能
- ブラウザ内自動保存に対応
- 配置ファイル `.choruslayout` として保存・読み込み可能
- 運営者側のスプレッドシートやサーバーに名簿・配置データを保存しない公開版仕様

## 個人情報の扱いについて

この公開版では、入力された氏名・パート・配置データを、運営者側のスプレッドシートやサーバーには保存しません。

データは主に以下の場所に保存されます。

- 利用者のブラウザ内の一時保存領域
- 利用者が自分で保存した `.choruslayout` ファイル
- 利用者が自分で保存した CSV / TSV ファイル
- 利用者が出力した PNG / PDF ファイル

そのため、大事な配置はブラウザ内保存だけに頼らず、`.choruslayout` ファイルとして保存しておくことをおすすめします。

## 使い方

### 1. 名簿が既にある場合

1. 「名簿を貼り付けて読み込み」または「CSV/TSVを読み込み」を使います。
2. 氏名とパートの一覧を読み込みます。
3. 配置案を作成します。
4. 候補から使いたい案を選びます。
5. キャンバス上でカードを並べ替えます。
6. PNG画像、印刷/PDF、配置ファイルとして保存します。

名簿の例：

```csv
氏名,パート
山田花子,S
佐藤明子,A
鈴木一郎,T
田中太郎,B
```

タブ区切りや、Excel・Googleスプレッドシートからコピーした表の貼り付けにも対応しています。

### 2. 各パートの人数だけ分かっている場合

1. パート構成を選びます。
2. 各パートの人数を入力します。
3. 配置案を作成します。
4. 配置を調整します。
5. 必要に応じて、後から名前入力用CSVを出力します。
6. CSVに名前を入力し、アプリに読み込んで反映します。

### 3. 配置を保存する

作成した配置は、以下の方法で保存できます。

- ブラウザ内保存
- `.choruslayout` ファイルとして保存
- PNG画像として保存
- ブラウザ印刷からPDF保存

別の端末でも使いたい場合や、長期間保存したい場合は、`.choruslayout` ファイルとして保存してください。

## PCでの操作

- カードをドラッグ：移動・差し込み
- ダブルクリック：カード編集
- 空白部分をドラッグ：範囲選択
- Ctrl / Cmd + クリック：複数選択
- 右クリック：追加・編集メニュー

## スマートフォンでの操作

- 通常ドラッグ：キャンバス移動
- ピンチイン / ピンチアウト：拡大・縮小
- カードを長押ししてドラッグ：カード移動・差し込み
- カードをダブルタップ：カード編集
- 複数選択ON：複数カードを選択してまとめて移動
- 未配置トレイ：画面下部から開閉

## 公開方法

### 自分のホームページで公開する場合

`index.html` を、自分のWebサーバーやホームページスペースにアップロードしてください。

例：

```text
public_html/
└─ chorus-stage-layout/
   └─ index.html
```

その場合、次のようなURLで利用できます。

```text
https://example.com/chorus-stage-layout/
```

### Google Apps Scriptで公開する場合

GASで公開したい場合は、`Code.gs` と `Index.html` をApps Scriptプロジェクトに配置します。

公開版では、GAS側は基本的にWebページを表示するだけです。  
スプレッドシートへの名簿保存や配置保存は行いません。

Webアプリとしてデプロイする場合の設定例：

```text
次のユーザーとして実行：自分
アクセスできるユーザー：全員
```

Googleログインなしで使わせたい場合は、利用しているGoogleアカウントや組織の設定で外部公開が許可されている必要があります。

## ローカルで使う場合

`index.html` をブラウザで開くだけでも利用できます。

ただし、ブラウザや端末の設定によっては、一部のファイル読み込み・保存機能の挙動がWeb公開時と異なる場合があります。  
安定して使う場合は、GitHub Pagesや自分のWebサーバーで公開する方法をおすすめします。

## 注意事項

- 実名入りのCSV、`.choruslayout`、Excelファイルなどを公開リポジトリにアップロードしないでください。
- ブラウザ内保存は、同じ端末・同じブラウザでのみ有効です。
- ブラウザのデータ削除、シークレットモード、端末変更などにより、ブラウザ内保存データは失われることがあります。
- 重要な配置は、`.choruslayout` ファイルとして保存してください。
- 本アプリは合唱団内での配置検討を補助するためのツールです。最終的な配置判断は、会場条件、指揮者・演出上の要望、団員の事情などを踏まえて行ってください。
